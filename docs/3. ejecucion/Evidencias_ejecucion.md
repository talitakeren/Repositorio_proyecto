# Evidencias de Ejecución

## Introducción

Este documento presenta las evidencias del desarrollo e implementación del prototipo del Sistema de Generación Óptima de Horarios Académicos (SGOHA), incluyendo la aplicación de TDD y el funcionamiento de los principales módulos del sistema.

---

## 1. Implementación con TDD

Durante el desarrollo se aplicó la metodología **Test-Driven Development (TDD)** siguiendo el ciclo **Red → Green → Refactor**, permitiendo validar progresivamente las funcionalidades implementadas y garantizar el correcto funcionamiento del sistema.

## 2. Evidencias del Sistema

Las evidencias muestran el funcionamiento de los principales módulos del prototipo, incluyendo la gestión de docentes, disponibilidad horaria, cursos, aulas y la generación automática de horarios académicos.

## 3. Funcionamiento del Prototipo

Las evidencias permiten verificar:

- Gestión de cursos.
- Gestión de docentes.
- Gestión de aulas.
- Aplicación de restricciones académicas.
- Generación automática de horarios.
- Funcionamiento general del prototipo.

## Conclusión

Las evidencias presentadas demuestran la correcta implementación y funcionamiento del prototipo desarrollado para la generación óptima de horarios académicos.
